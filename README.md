# sashisu-realm

Sashisu Realm is a **full-stack anime merchandise e-commerce platform** built with a custom frontend and a Node.js + Express backend.  
The project focuses on **secure authentication, shopping cart flow, Stripe payments, and order management**, wrapped in a themed anime-inspired UI.

## features

### frontend

- responsive anime-themed UI (tailwind CSS)
- product browsing with categories and search
- shopping cart with quantity control
- order summary and confirmation pages
- authentication flow (signup & login)
- account and order history pages

### authentication

- secure signup and login
- token-based authentication
- session validation with protected routes

### payments

- stripe payment integration
- server-side payment intent creation
- secure checkout flow

### orders

- order creation after successful payment
- order history per user
- server-validated pricing (no client-side trust)
